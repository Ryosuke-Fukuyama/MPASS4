# README

## 開発言語
- Ruby version '2.6.5'
- rails version '6.0.5'

## 実行手順
```
$ git clone
$ cd rails_app
$ bundle
$ rails db:create db:migrate
$ rails db:seed
$ rails s

$ cd nuxt_app
$ yarn install
$ yarn dev
```
